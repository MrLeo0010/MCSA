# MCSA (Minecraft Server Analyser)

**MCSA** — это модульный инструмент на Go, предназначенный для сканирования, фильтрации и анализа игровых серверов Minecraft. Проект позволяет автоматизировать процесс проверки — от массового отсеивания открытых портов до детального разбора конфигурации конкретного сервера.

> ⚠️ **Замечание:** Проект находится в стадии активной разработки. Документация может немного отставать от актуального состояния кода, однако общая концепция и структура всегда остаются верными.

## Модули системы

### 1. MCSA Minimal (Масс-фильтр)
Высокоскоростной инструмент для обработки больших списков IP-адресов и портов (например, после Masscan или Advanced Port Scanner).
* Работает в многопоточном режиме (Worker Pool).
* Опрашивает порты по протоколу Minecraft и отсеивает сторонние сервисы (SSH, HTTP, RDP).
* Сохраняет список валидных игровых серверов в отдельный файл.

### 2. MCSA Base (Базовый опрос)
Консольная утилита для быстрого одиночного сканирования конкретного сервера.
* Получает информацию через Server List Ping.
* Выводит пинг, версию протокола, текущий MOTD, онлайн и выборку игроков.

### 3. MCSA Extended (Глубокий анализ)
Модуль для детальной разведки параметров входа на сервер.
* Использует интеграцию с изолированным клиентом *Minecraft Console Client (MCC)*.
* Анализирует системные сообщения и чат в режиме реального времени, подсвечивая найденные триггеры.
* Автоматически определяет тип авторизации: **Лицензия** (требуется сессия Microsoft), **Пиратка** (требуется регистрация/логин) или **Свободный вход**.

## Вспомогательные инструменты

В комплекте с проектом поставляется Python-скрипт для предварительной обработки данных. Он очищает «грязный» вывод Advanced Port Scanner, извлекает номера портов с помощью регулярных выражений и форматирует их в вид `IP:PORT` для последующей загрузки в `MCSA Minimal`.

## Сборка проекта

Компиляция всех модулей под нужную платформу осуществляется с помощью готовых скриптов автоматизации:

* **Windows:** `build_win.bat`
* **Linux:** `build_linux.sh`
